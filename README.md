# State Department

US Department of State APIs.json provider profile covering public REST APIs for consular affairs, travel advisories, passport services, and country information.

**Base URL:** https://cadataapi.state.gov  
**Website:** https://www.state.gov/  
**Travel Portal:** https://travel.state.gov/  
**GitHub:** https://github.com/USStateDept  

## APIs

- **Consular Affairs API** - Travel advisories, country travel information, passport locations, issuance statistics
- **Travel Advisories** - All countries, four alert levels, real-time updates

## Authentication

No authentication required. All endpoints are open and public.

## Rate Limits

No published rate limits. Use politely with caching.
